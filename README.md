# Stock Market Prediction System

## Overview

This project is a machine learning-based stock market prediction platform that forecasts stock trends using historical market data. Multiple machine learning models were implemented and compared to evaluate prediction performance.

## Features

* Stock price prediction using LSTM
* Stock price prediction using GRU
* Random Forest model implementation
* Model comparison and evaluation
* Interactive web-based user interface
* Visualization of prediction results
* Confusion matrix analysis

## Technologies Used

### Backend

* Python
* TensorFlow / Keras
* Scikit-Learn
* Pandas
* NumPy

### Frontend

* React
* Vite
* JavaScript
* HTML/CSS

## Project Structure

* Backend contains machine learning models, training scripts, and API logic.
* Frontend contains the React-based user interface.
* Screenshots folder contains project outputs and model evaluation results.

## Models Implemented

1. LSTM (Long Short-Term Memory)
2. GRU (Gated Recurrent Unit)
3. Random Forest
